CORE-MOBILE-APP-Reporting — Bootstrap Data Flow

Reader → Column Mapping → Applier · Luồng đọc data từ SQL Server, ánh xạ schema, và ghi vào PostgreSQL

I. Tổng quan Pipeline

Toàn bộ bootstrap data flow là một pipeline 3 tầng:

1. BootstrapSyncService.ExecuteAsync
├─ Acquire advisory lock (pg_try_advisory_lock)
│ └─ [Lock busy] → SkippedLocked
├─ reader.ReadAsync(config)
│ ├─ Mở SQL Server ReadCommitted transaction
│ ├─ CHANGE_TRACKING_CURRENT_VERSION() → baseline
│ ├─ SELECT data với READPAST hint
│ ├─ Verify version → nếu changed → rollback + retry (max 3)
│ └─ return BootstrapSnapshot(baseline, rows)
├─ applier.ApplyBootstrapAsync(config, snapshot)
│ ├─ 1 PostgreSQL transaction
│ ├─ BuildTargetValues(rule, row)
│ ├─ Upsert all rows
│ ├─ Deactivate orphan rows
│ └─ Upsert checkpoint
├─ runLog.WriteAsync(...)
└─ return SyncRunResult

📍 Application/Features/CentralDbSync/Services/BootstrapSyncService.cs:73-136

Atomicity guarantee: Hoặc toàn bộ upsert + orphan deactivate + checkpoint được commit trong 1 transaction PostgreSQL, hoặc không gì thay đổi. Snapshot consistency được đảm bảo bởi version verify retry loop ở reader.

3.1 Build Bootstrap SELECT

SqlServerSqlBuilder.BuildBootstrapSelect(rule) — sinh SELECT từ ColumnMapping list

3.2 Consistent Snapshot — Version Verify Retry

Đảm bảo snapshot data và baseline checkpoint version khớp nhau:

ReadAsync
├─ 1. BEGIN TRAN (ReadCommitted)
├─ 2. baseline = CHANGE_TRACKING_CURRENT_VERSION()
├─ 3. SELECT data
├─ 4. versionAfter = CHANGE_TRACKING_CURRENT_VERSION()
├─ if (baseline == versionAfter)
│ └─ COMMIT → return snapshot
└─ else → ROLLBACK → retry (max 3 lần) 
// Infrastructure/CentralDbSync/SqlServerGenericReader.cs:30-68
for (var attempt = 1; attempt <= MaxBootstrapRetries; attempt++)
{
    await using var tx = await conn.BeginTransactionAsync(
        IsolationLevel.ReadCommitted, ct);

    var baseline = await conn.ExecuteScalarAsync<long>(
        "SELECT CHANGE_TRACKING_CURRENT_VERSION()", transaction: tx);

    var rows = await ReadRowsAsync(conn, tx, select, ct);

    var versionAfter = await conn.ExecuteScalarAsync<long>(
        "SELECT CHANGE_TRACKING_CURRENT_VERSION()", transaction: tx);

    if (baseline == versionAfter)
    {
        await tx.CommitAsync(ct);
        return new BootstrapSnapshot(baseline, rows);
    }
    await tx.RollbackAsync(ct);
}
throw new InvalidOperationException(
    "Failed to capture consistent snapshot after 3 attempts.");

IV. Applier — Ghi vào PostgreSQL

PostgresGenericApplier

4.1 BuildTargetValues — Ánh xạ field

Mỗi row trong snapshot được ánh xạ từ source row (keyed bằng alias SQL) sang target values (keyed bằng PostgreSQL column name)

BuildTargetValues(rule, sourceRow)
├─ foreach column in rule.Columns
│ ├─ IsActiveFlag? → evaluate ActivePredicate
│ ├─ Transform? → resolve & apply transformer
│ ├─ SourceColumn? → sourceRow.GetValueOrDefault(column.SourceColumn)
│ └─ else → sourceRow.GetValueOrDefault(column.TargetColumn)
└─ return dict{partner_id: 1, company_id: 100, ..., is_active: true} 
// Infrastructure/CentralDbSync/PostgresGenericApplier.cs:205-227
private Dictionary<string, object?> BuildTargetValues(
    TableMappingRule rule, GenericSourceRow sourceRow)
{
    var values = new Dictionary<string, object?>(
        StringComparer.OrdinalIgnoreCase);
    foreach (var column in rule.Columns)
    {
        PostgresTypeMap.EnsureSupported(column.TargetType);
        values[column.TargetColumn] = ResolveColumnValue(rule, column, sourceRow);
    }
    return values;
}

private object? ResolveColumnValue(TableMappingRule rule,
    ColumnMapping column, GenericSourceRow sourceRow)
{
    if (column.IsActiveFlag)
        return IsActive(rule, sourceRow);  // evaluate ActivePredicate
    if (!string.IsNullOrWhiteSpace(column.Transform))
        return transformerRegistry.Resolve(column.Transform)
            .Transform(sourceRow.Values);
    if (!string.IsNullOrWhiteSpace(column.SourceColumn))
        return sourceRow.GetValueOrDefault(column.SourceColumn);
    return sourceRow.GetValueOrDefault(column.TargetColumn);
}

4.2 ActivePredicate — Tính is_active

IsActive kiểm tra tất cả predicates — nếu tất cả đều true → row được coi là active. VD với CRM.Partners: IsCustomer == true

// Infrastructure/CentralDbSync/PostgresGenericApplier.cs:229-254
private static bool IsActive(TableMappingRule rule, GenericSourceRow sourceRow)
{
    if (rule.Source.ActivePredicate.Count == 0)
        return true;

    return rule.Source.ActivePredicate.All(predicate
        => Evaluate(predicate, sourceRow));
}

private static bool Evaluate(ColumnPredicate predicate, GenericSourceRow sourceRow)
{
    var actual = sourceRow.GetValueOrDefault(predicate.Column);
    return predicate.Operator switch
    {
        PredicateOperator.Eq => Equals(actual, predicate.Value),
        PredicateOperator.Neq => !Equals(actual, predicate.Value),
        PredicateOperator.IsNull => actual is null,
        // ... Gt, Gte, Lt, Lte, In, NotIn, IsNotNull
    };
}

4.3 Upsert SQL

UpsertSqlBuilder.BuildUpsert(rule) sinh INSERT ON CONFLICT DO UPDATE tự động từ ColumnMapping:

// Infrastructure/CentralDbSync/Sql/UpsertSqlBuilder.cs:7-20
public string BuildUpsert(TableMappingRule rule)
{
    var insertColumns = rule.Columns
        .Select(c => QuotePgIdentifier(c.TargetColumn));
    var valueColumns = rule.Columns
        .Select(c => "@" + c.TargetColumn);
    var conflictColumns = rule.Target.PrimaryKey
        .Select(QuotePgIdentifier);
    var updateColumns = rule.Columns
        .Where(c => !c.IsPrimaryKey)
        .Select(c => $"{QuotePgIdentifier(c.TargetColumn)} = EXCLUDED.{QuotePgIdentifier(c.TargetColumn)}")
        .Append($"{QuotePgIdentifier("synced_at")} = NOW()");

    return $@"INSERT INTO {QuotePgTable(rule.Target.Schema, rule.Target.Table)}
({string.Join(", ", insertColumns)})
VALUES ({string.Join(", ", valueColumns)})
ON CONFLICT ({string.Join(", ", conflictColumns)})
DO UPDATE SET {string.Join(", ", updateColumns)}";
}

4.4 ApplyBootstrapAsync — Toàn bộ logic

ApplyBootstrapAsync(config, snapshot)
├─ BEGIN TRAN
├─ 1. Upsert all rows
│ ├─ foreach row in snapshot.Rows
│ │ ├─ values = BuildTargetValues(rule, row)
│ │ │ └─ source: {partner_id=1, company_id=100, ...}
│ │ │ └─ → target: {partner_id=1, is_active=true, ...}
│ │ ├─ conn.ExecuteAsync(upsertSql, ToDynamicParameters(values))
│ │ └─ snapshotPks.Add(values[targetPrimaryKey])
│ └─ → upsertedCount
├─ 2. Deactivate orphans
│ ├─ BuildLifecycleOrphans(rule, "sourceSystem")
│ ├─ -- UPDATE report.partners SET is_active = false
│ ├─ -- WHERE source_system = 'erp' AND partner_id <> ALL(@snapshotPks)
│ └─ → orphanLifecycleCount
├─ 3. Upsert checkpoint
│ ├─ -- INSERT INTO sync_meta.checkpoint (...) VALUES (...)
│ └─ -- ON CONFLICT DO UPDATE SET ...
├─ COMMIT
└─ return SyncRunResult { Outcome = Succeeded, ... } 
// Infrastructure/CentralDbSync/PostgresGenericApplier.cs:124-203
async Task<SyncRunResult> ApplyBootstrapAsync(
    TableSyncConfig config, BootstrapSnapshot snapshot, CancellationToken ct)
{
    var rule = ruleProvider.Get(config.SourceTable);
    await using var conn = new NpgsqlConnection(connectionString);
    await conn.OpenAsync(ct);
    await using var tx = await conn.BeginTransactionAsync(ct);

    try
    {
        var upsertSql = sqlBuilder.BuildUpsert(rule);
        var targetPrimaryKey = GetSingleTargetPrimaryKeyColumn(rule);
        var snapshotPrimaryKeys = new List<object?>(snapshot.Rows.Count);
        var upsertedCount = 0;

        // Step 1: Upsert all rows
        foreach (var row in snapshot.Rows)
        {
            var values = BuildTargetValues(rule, row);
            snapshotPrimaryKeys.Add(values[targetPrimaryKey.TargetColumn]);
            await conn.ExecuteAsync(upsertSql,
                ToDynamicParameters(values), transaction: tx);
            upsertedCount++;
        }

        // Step 2: Deactivate orphans
        var snapshotPks = BuildTypedPrimaryKeyArray(
            targetPrimaryKey, snapshotPrimaryKeys);
        var hasActiveFlag = sqlBuilder.HasActiveFlag(rule);
        await conn.ExecuteAsync(
            sqlBuilder.BuildLifecycleOrphans(rule, "sourceSystem"),
            new { sourceSystem = rule.OwnershipScope, snapshotPks },
            transaction: tx);

        // Step 3: Upsert checkpoint
        await conn.ExecuteAsync($@"INSERT INTO {CheckpointTable}
            (source_table, last_sync_version, sync_status, ...)
            VALUES (...) ON CONFLICT (source_table) DO UPDATE SET ...",
            new { baselineVersion = snapshot.BaselineVersion, ... },
            transaction: tx);

        await tx.CommitAsync(ct);
        return new SyncRunResult { Outcome = Succeeded, ... };
    }
    catch { await tx.RollbackAsync(ct); throw; }
}

Bootstrap Data Pipeline — Full Flow

[SQL Server ERP]
│
├─► SqlServerGenericReader (ReadCommitted + READPAST)
│ ├─ capture CHANGE_TRACKING_CURRENT_VERSION()
│ ├─ SELECT theo SqlServerSqlBuilder (build từ ColumnMapping)
│ └─ version verify → fail → retry (max 3)
│
├─► BootstrapSnapshot
│ ├─ BaselineVersion = CT version
│ └─ Rows = GenericSourceRow[] (Dictionary<alias, object?>)
│
├─► PostgresGenericApplier (1 PG transaction)
│ ├─ BuildTargetValues — map field từng row
│ │ ├─ SourceColumn → copy value
│ │ ├─ IsActiveFlag → evaluate ActivePredicate
│ │ └─ Transform → IValueTransformer
│ ├─ Upsert — INSERT ON CONFLICT DO UPDATE
│ ├─ Deactivate orphans — WHERE PK NOT IN snapshot

└─► report.partners / units / sizes
